NeuroAI

NeuroAI is a lightweight platform for automated brain MRI analysis using deep learning. Users upload a NIfTI (`.nii.gz`) file, which is rendered immediately and sent to a GPU-powered segmentation model for processing. The backend API manages communication with the model and polls asynchronously until the prediction is ready. Once complete, the tumor mask is retrieved and overlaid on the original scan for direct visual comparison. The entire system is modular and deployable on minimal infrastructure, including static web hosting.
